# Архив документации

Этот каталог предназначен для документов, которые были сняты с актуальности, но ещё должны храниться рядом с репозиторием.

Типичные кандидаты:
- старые `docs/test-matrix/*`;
- ранний `docs/trusttunnel-v1.md`;
- импортированные черновики после ручной сверки;
- промежуточные служебные заметки, которые больше не должны участвовать в current-слое.

Правила:
- сюда попадают только документы, уже вытесненные из `docs/current/*`;
- архив не является source of truth для текущего состояния;
- перенос в архив не должен ломать `docs/history/*`;
- при переносе в архив нужно оставлять запись о том, какой current-документ его заменил;
- возвращать документ из архива в current-слой можно только при наличии более новых и проверяемых доказательств, а не из-за самого факта существования старого текста.
