## Scope Of Public Docs

Public docs in this repository are limited to:

- what the fork is
- what feature surface it exposes
- how to write supported configs
- what combinations are intentionally unsupported

Environment-specific runbooks, development notes, internal validation logs, and private deployment details are intentionally kept out of the public documentation set.
